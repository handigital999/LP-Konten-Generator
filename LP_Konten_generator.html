FAQ

MASIH ADA
PERTANYAAN?

Apakah saya perlu punya akun OpenAI atau API key sendiri? +
Ya, kamu perlu akun OpenAI dengan API key untuk menjalankan bot ini karena biaya generate gambar dikenakan per request ke API OpenAI. Saya akan bantu kamu setup akun dan API key-nya. Biaya API OpenAI sangat terjangkau — sekitar $0.04–0.08 per gambar (Rp600–1.200/gambar).
Apakah perlu VPS atau server? +
Ya, perlu VPS untuk menjalankan n8n workflow secara 24/7. VPS lokal bisa dapat dengan harga Rp50–100rb/bulan. Saya bantu setup-nya sekalian, jadi kamu tidak perlu repot teknis.
Seberapa bagus kualitas gambar yang dihasilkan? +
Menggunakan model GPT-Image-1 dari OpenAI — ini salah satu model image generation terbaik saat ini. Resolusi bisa sampai 1792x1024 pixel, siap dipakai untuk posting sosmed, cetak, atau iklan digital tanpa perlu diedit.
Berapa lama proses setup hingga bot bisa dipakai? +
Biasanya 1–2 hari setelah semua akses diberikan. Meliputi setup VPS, instalasi n8n, konfigurasi Telegram Bot, dan koneksi ke OpenAI API. Kamu akan dapat panduan lengkap cara pakainya.
Apakah gambar yang dihasilkan bebas copyright? +
Gambar yang dihasilkan menggunakan API OpenAI. Berdasarkan kebijakan OpenAI, kamu memiliki hak atas gambar yang dibuat dan bisa digunakan untuk keperluan komersial. Namun tetap disarankan untuk tidak menyalin gaya artis spesifik atau karakter berhak cipta.
Bisa tidak kalau saya mau request fitur tambahan? +
Bisa! Workflow n8n sangat fleksibel untuk dikustomisasi. Misalnya tambah fitur ukuran gambar yang bisa dipilih, style preset, watermark otomatis, atau integrasi ke platform lain. Diskusikan kebutuhanmu dan saya berikan estimasi biaya tambahan-nya.
